# Portfolio Website

A modern, elegant GenAI & Risk Analytics portfolio built with Next.js and TypeScript.

## Quick Start

```bash
npm install
npm run dev
```

Visit http://localhost:3000

## Build & Deploy

```bash
npm run build
```

Static output in `out/` folder. GitHub Actions auto-deploys on push.

## Customize

Edit `src/data/portfolio.ts` to update your profile.
